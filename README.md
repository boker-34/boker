Control remoto para TV LG con sistema operativo Netcast.

Forma de uso:

```
python ./kif.py -i IP_TELEVISOR -k KEY_DE_EMPAREJAMIENTO
```

Para realizar un ataque de fuerza bruta:

```
python ./kif.py -i IP_TELEVISOR
```
